Unity3D游戏开发预备课(二)

Unity3D

碰撞器和触发器的概念及相关处理事件


碰撞器

碰撞器概念

  1. 在U3D引擎中任意物体之间的碰撞事件的产生都依赖于碰撞器，即“Collider”。
  2. Object之间的碰撞实际上是“Collider”之间的碰撞。
  3. Object所具有的网格渲染器，即”Mesh Renderer”并不参与碰撞的检测，仅仅是展现给人看的外观。

碰撞器所具有的属性

  • Material 材质

  • Is Trigger 是否触发器

  • Radius 半径(胶囊形或圆形碰撞器)

  • Center 中心

  • Size 大小 在X、Y、Z三维上的碰撞器尺寸。

碰撞器事件

实例

只要为两个Object设置了碰撞器，就会继续碰撞检测。

触发器

触发器概念

  1. 勾选碰撞器的”Is Trigger”属性,碰撞器就会变成触发器。
  2. 即原来具有这个碰撞器的Object现在不能发生碰撞。
  3. 触发器具有新的触发事件，在游戏中可以用来表示在特定的区域中发生的事件，如在毒雾中持续削减HP。

触发器所具有的属性

  • Material 材质

  • Is Trigger 是否触发器

  • Radius 半径(胶囊形或圆形触发器)

  • Center 中心

  • Size 大小 在X、Y、Z三维上的碰撞器尺寸。
